# Lightweight Cost-Aware Web Caching Design: Evaluation Against Generic Policies

This repository contains the code for the Bachelor Thesis exploring **how lightweight and cost-aware caching algorithm design decisions impact processing cost-efficiency compared to conventional web cache replacement policies.**

The main evaluation component is the `policy_runner.py` script, which implements the Policy Runner framework. This framework allows for the evaluation of various caching policies, including both conventional and cost-aware algorithms, against a common set of web request traces.
